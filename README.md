# ¿Cómo se dice...?

为中英日三语使用者设计的西班牙语课程。

## 故事

Jack——一个会说中文、英文、日文的海盗——离开加勒比多年后，西班牙语忘得一干二净。他的朋友 Carlos 帮他重新捡起来。这是他们的对话记录。

## 形式

探索日志式的对话体，不是传统教材。每个词都从最接近的已知语言切入——日语帮发音，中文帮辅音，英语帮词汇。带 TTS 语音（Web Speech API）。

## 课程

1. ¡Hola! — 第一声、发音、问候 ✅
2. Ponme un café — 在酒馆点东西，祈使句
3. ¿Cómo? — 听不懂时怎么活下来
4. Me he perdido — 问路、方位、现在完成时
5. ¿Qué nos recomiendas? — 餐厅吃饭，ser vs estar
6. La silla — 名词性别，第一堵墙
7. ¿Quedamos? — 打电话约人，代词系统
8. Estoy aquí — 表达情绪，说不出口的话
9. ¿Qué pasó? — 讲故事，过去时的两张面孔
10. La llamada — 打电话给女儿，谈未来
11. Ojalá — 虚拟语气，最高的墙
12. Bienvenidos — 闲聊、开玩笑、像个人

## 技术

Pure HTML/CSS/JS, no framework. TTS via Web Speech API. Hosted on GitHub Pages.
